# 📊 E-commerce Performance Analytics Dashboard

## 📌 Project Overview

Built an end-to-end **E-commerce Performance Analytics Dashboard** to analyze **sales performance, customer behavior, product trends, and regional insights** using Python, SQL, and Power BI.

The dashboard is divided into **three analytical views**:

- 📈 Sales Analytics  
- 👥 Product & Customer Analytics  
- 🌍 Regional & Review Analytics  


# 🎯 Project Objective

To analyze e-commerce performance and identify:

- Revenue trends  
- Customer behavior  
- Product performance  
- Regional performance  
- Customer satisfaction drivers

## 🛠 Tech Stack

- Python (Pandas, NumPy)
- SQL
- Power BI
- Excel


## 📂 Project Files

- `ecom_cleaning.ipynb` — Data Cleaning (Python)
- `ecomm_analysis.sql` — Business Analysis (SQL)
- `ecom_dashboard.pbix` — Power BI Dashboard

# 📈 Key Insights

## 💰 Sales Insights

- Generated **$15.5M total revenue** across **100K+ orders**
- Revenue showed **steady growth from 2016 to 2018**
- **Average Order Value ≈ $160**
- **Credit Card payments accounted for ~79%** of transactions



## 📦 Product Insights

- **Top 10 product categories contributed majority of revenue**
- Some categories showed **high order volume but low revenue**
- Premium categories had **higher AOV but lower order volume**



## 👥 Customer Insights

- **98% customers were new customers**
- Repeat customers showed **higher average order value**
- Customer demand concentrated in **few major regions**



## 🌍 Regional Insights

- Revenue concentrated in **top-performing states**
- Some regions showed **longer delivery times**
- High-performing regions showed **higher customer satisfaction**



## ⭐ Review & Delivery Insights

- **Average review score ≈ 4.5**
- **Faster delivery correlated with higher review scores**
- Slow delivery regions showed **lower customer satisfaction**
- Few categories consistently received **higher ratings**
